# Early-life-HFHS_Feeding_Microbiota

This repository contains all the information (raw data, codes, figures in svg, and outputs) for the manuscript "The enduring effects of early-life high-fat/high-sugar exposure on feeding behavior are sensitive to _Bifidobacterium_". 

The raw and processed molecular and behavioral data can be found in: https://github.com/CristinaCuesta-Marti/earlylifeHFHS_feeding_microbiota_2025/tree/data
The output files, including the statistical analysis, can be found in: https://github.com/CristinaCuesta-Marti/earlylifeHFHS_feeding_microbiota_2025/tree/outputs
The files with the information for outliers can be found in: https://github.com/CristinaCuesta-Marti/earlylifeHFHS_feeding_microbiota_2025/tree/outliers
The code (scripts) can be found in: https://github.com/CristinaCuesta-Marti/earlylifeHFHS_feeding_microbiota_2025/tree/scripts
The svg files for main and supplementary figures can be found in: https://github.com/CristinaCuesta-Marti/earlylifeHFHS_feeding_microbiota_2025/tree/figures
Additional information for fecal microbiome 16S data and analysis can be found in: https://github.com/CristinaCuesta-Marti/earlylifeHFHS_feeding_microbiota_2025/tree/16S-dada2-outputs
